1. Einleitung

Die Taktische Karte ist ein Dashboard zur Live-Darstellung von Ereignissen auf einer interaktiven Karte. Sie wurde für Einsatzkräfte, Journalisten und Lagezentren entwickelt.

FunktionBeschreibung
🗺 Live-KarteInteraktive Karte mit Markern, Routen und Events
✏️ Admin-PanelDaten eingeben per Klick auf die Karte
🎯 GeländetaufeGebäude markieren mit Uhrzeit-Positionen
🤖 KI-UpdateAutomatische News-Recherche (optional)
📴 OfflineVollständig ohne Internet nutzbar
Voraussetzung: Nur Python 3.11+ (kostenlos). Alle anderen Abhängigkeiten sind im Paket enthalten.

2. Installation

1 ZIP-Datei taktische-karte-v1.0.zip entpacken
2 Terminal im entpackten Ordner öffnen
3 python server.py eingeben
4 Browser öffnen: http://localhost:8765

3. Erste Schritte

3.1 Die Oberfläche

Nach dem Öffnen siehst du:

  • Links: Die interaktive Karte (OSM-Standard)
  • Rechts: Die Seitenleiste mit Bedienelementen

3.2 Karten-Layer wechseln

In der Seitenleiste unter 🗺 KARTE:

  • Topo — Straßenkarte (Standard)
  • Sat — Satellitenbilder
  • Hybrid — Satellit + Straßennamen

3.3 Overlays ein-/ausschalten

  • ◆ Blockaden — Alle Blockade-Marker
  • ▲ Events — Ereignisse und Vorfälle
  • ━ Sperrungen — Gesperrte Straßenabschnitte

4. Die Kartenansicht

4.1 Navigation

  • Mausrad — Rein-/Rauszoomen
  • Ziehen — Karte verschieben

4.2 Marker

Klicke auf einen Marker für Details:

  • Typ (Großblockade, Sitzblockade, etc.)
  • Zeitstempel und Notizen

5. Admin-Panel — Daten eingeben

Öffne http://localhost:8765/admin.html

5.1 Marker hinzufügen

1 Tab ➕ Marker wählen
2 Auf die gewünschte Position auf der Karte klicken
3 Formular ausfüllen: Name, Typ, Teilnehmerzahl, Notiz
4 💾 Speichern

5.2 Einträge verwalten

  • 📋 Liste: Alle Einträge anzeigen, anklicken zum Bearbeiten
  • 🗑: Im Admin-Panel löschen

5.3 Export / Import

Tab 📤 Export:

  • 📋 Kopieren: JSON in die Zwischenablage
  • 💾 Download: Als Datei speichern
  • 📥 Import: JSON einfügen, um alte Daten zu laden

6. Geländetaufe

Die Geländetaufe markiert Gebäude mit Uhrzeit-Positionen (0/3/6/9 Uhr) — für die taktische Orientierung im Gelände.

1 Tab 🎯 Taufen wählen
2 Satelliten-Ansicht aktivieren (Taste T)
3 Auf das gewünschte Gebäude klicken
4 Uhrzeit wählen — die Blickrichtung zum Objekt:
UhrRichtungFarbe
0Norden ↑Gelb
3Osten →Grün
6Süden ↓Blau
9Westen ←Pink
5 Bezeichnung + Notiz eingeben → 💾 Speichern
Tipp: Taufen überleben Browser-Refresh und Neustart. Sie werden in localStorage gespeichert.

7. KI-Auto-Update (optional)

Mit Hermes Agent kann die Karte automatisch aktualisiert werden:

1 Hermes installieren: pip install hermes-agent
2 Cronjob-Prompt aus cronjob-prompt.txt kopieren
3 Platzhalter ersetzen (Ort, Hashtag, Datum)
4 In Hermes einrichten: /cron create "*/5 * * * *"
Der Cronjob sucht alle 5 Minuten:
  → Polizei-Meldungen
  → X/Twitter (#Hashtag)
  → Presseportal
  → IndyMedia

Er schreibt die Daten in blockaden-data.json.
Die Karte lädt automatisch nach.
Wichtig: Der Cronjob läuft nur, wenn Hermes läuft. Für Dauerbetrieb: hermes gateway install

8. Offline-Betrieb mit Tiles

Für den Einsatz ohne Internet können Kartendaten vorab heruntergeladen werden.

8.1 Setup-GUI öffnen

http://localhost:8765/setup.html

8.2 Konfiguration

  • Bundesland wählen (Zoom 10–16)
  • OSM / Satellit — Kartentypen wählen
  • Hotspots — Rechteck auf Karte ziehen für Zoom 17–19 (Gebäude-Ansicht)

8.3 Download starten

▶ Button klicken. Der Fortschritt wird live angezeigt.

BundeslandGröße (Z10–16)
Thüringen~400 MB
Bayern~1,6 GB
Berlin~50 MB
Achtung: Der Download startet den Tile-Server auf Port 8766. Stelle sicher, dass tile_server.py läuft.

9. Export & Import

9.1 Daten exportieren

Im Admin-Panel → Tab 📤 Export:

  • 📋 Kopieren — JSON in Zwischenablage
  • 💾 Download — Als .json-Datei speichern

9.2 Daten importieren

📥 Import-Button → JSON einfügen. Überschreibt aktuelle Blockaden, Events und Taufen.

9.3 Release bauen

python build.py
# → dist/taktische-karte-v1.0.zip (mit allen Daten)
